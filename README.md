# Ars-Magica-Open-License
The text of Ars Magica 5th Edition and its sourcebooks (53 books total) has been released under an open license: the Creative Commons Attribution-ShareAlike 4.0 International license (CC BY-SA 4.0): https://www.atlas-games.com/arsmagica/openars

This is a work-in-progress repository for extracted and doctored markdown of the Open Licensed material.
